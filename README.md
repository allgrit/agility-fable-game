# 🐕 Agility Trial!

Аркадная браузерная игра про собачий спорт **аджилити**. Ты — собака на соревновательной
трассе: хендлер бежит рядом и выкрикивает команды, а твоя задача — правильно и вовремя
на них реагировать (QTE). Точность реакции определяет скорость, чистоту прогона и результат.

**Играть:** https://allgrit.github.io/agility-fable-game/

## Особенности

- **Реальные трассы чемпионатов** — режим «Чемпионат мира» содержит 6 трасс, оцифрованных
  с официальных судейских карт: Nordic Championship 2018 и FCI Grade 3 (судья Anders Virtanen),
  4 раунда AKC National Agility Championship 2021, включая финал (источники в `research/`).
- **Процедурный генератор трасс** — режим «Карьера»: Novice → Open → Excellent → Masters,
  трассы строятся по правилам FCI/AKC (слалом ×1, контактные снаряды, лимиты типов, дистанции 5–8 м).
- **Аутентичное судейство** — SCT (standard course time), 5 faults за сбитую планку и пропуск
  контактной зоны, отказы (refusals), time faults, чистый прогон = квалификация «Q».
- **10 типов снарядов** — барьеры, шина, стена, длинный прыжок, туннель, слалом (12 стоек),
  горка, бум, качели, стол — каждый со своей QTE-механикой.
- **3 породы** — бордер-колли (скорость), шелти (широкое окно реакции), джек-рассел (комбо).
- Полностью синтезированный звук (WebAudio, ни одного аудиофайла), процедурные анимации,
  частицы, screen shake, hit-stop, толпа с волной оваций.

## Управление

| Клавиша | Действие |
|---|---|
| `ПРОБЕЛ` | Прыжок (барьер, шина, стена, длинный прыжок) |
| `↓` | Туннель |
| `← →` попеременно | Слалом (в ритме!) |
| `↑` удерживать, отпустить в жёлтой зоне | Горка, бум (контактная зона!) |
| `↑`, затем `ПРОБЕЛ` в момент опускания | Качели |
| `ПРОБЕЛ` удерживать | Стол (выдержка) |

## Разработка

Ванильный JS + Canvas 2D + WebAudio, ноль зависимостей.

```bash
python -m http.server 8734      # локальный запуск (нужен из-за ES-модулей)
node --test tests/logic.test.mjs # тесты чистой логики (генератор, QTE, скоринг, реальные трассы)
```

Архитектура: чистая логика (`course.js`, `qte.js`, `scoring.js`, `spline.js`) отделена
от рендера и звука — тестируется в node без браузера. Детали в `docs/superpowers/specs/`.
